# Muhammad Ahmad — Backend Developer Portfolio (v2)

Personal portfolio showcasing backend engineering skills: NestJS, TypeScript, PostgreSQL, and REST API development.

## Stack

- HTML5, CSS3, JavaScript (ES6+)
- Bootstrap 5.2
- Font Awesome 6

## Structure

```
├── index.html
├── EXECUTIVE-SUMMARY.md    ← Upgrade spec & checklist
├── CSS/style.css
├── Javascript/
│   ├── portfolio.js
│   └── Muhammad Ahmad CV.pdf
└── Images/
```

## Run locally

Open `index.html` in a browser, or use a local server:

```bash
npx serve .
```
